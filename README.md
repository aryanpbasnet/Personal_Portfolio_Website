# Aryan Basnet — Portfolio Website

This is my personal portfolio website, built to showcase my projects, skills, and experience as a developer and designer.  
It reflects my approach to clean UI, thoughtful interactions, and modern web development practices.

**Live Site:** https://aryanbasnet.com.np/

---

## Features

- **Dark / Light Mode Toggle**
- **Smooth animations & micro-interactions**
- **Custom cursor effect (desktop)**
- **Fully responsive design**
- **Project showcase with flip-card interaction**
- **Working contact form (EmailJS)**
- **Optimized & minified assets for performance**

---

## Built With

- **HTML5**
- **CSS3**
- **Vanilla JavaScript**
- **Font Awesome & Devicon**
- **AOS (Animate On Scroll)**
- **EmailJS**

No frameworks — built from scratch for full control and understanding.

---


---

## Deployment

The website is deployed as a static site and optimized for fast load times.  
Assets are minified and external libraries are loaded via CDN.
